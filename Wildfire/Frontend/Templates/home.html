Site Home Page

Welcome to the home page powered by Wildfire. You'll want to get this replaced with your own template, so here's how you go about it.

This page runs off a Twig template named home.html and a separate layout file layout.html To replace this page with your own layout and template you can do the following...

First add your local template directory to the application renderer. In Application.php boot method add: 

      
        $this->renderer->getLoader()->addPath(__DIR__."/Templates");

Then create the Templates directory and add your templates inside.
